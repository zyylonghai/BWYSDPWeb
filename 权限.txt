账户表：
(
UserId,账户号   主键
UserNm, 账户名 
Password,密码 （存的是密文）
Status,状态(已激活，未激活，冻结）

CreateDate, 创建日期
Creater,创建者
LastmodifyDate, 最后修改日期
LastmodifyUser, 最后修改者
)

角色表(主)：
(
     JoleId, 角色编号   主键
     JoleNm, 角色名
     Status, 状态（已激活，未激活，冻结）
     Remark, 备注
     
CreateDate, 创建日期
Creater,创建者
LastmodifyDate, 最后修改日期
LastmodifyUser, 最后修改者
)

角色（明细）:
(
    JoleId, 角色编号   主键
    ProgId, 功能ID  主键
    
CreateDate, 创建日期
Creater,创建者
LastmodifyDate, 最后修改日期
LastmodifyUser, 最后修改者

)
权限对象
(
     JoleId, 角色编号   主键
    ProgId, 功能ID  主键
    SeqNo, 项目号 主键（自动增长)
    ObjectType, 权限对象类型(1:操作对象 2:数据对象 )
    ObjectId, 对象ID（包括按钮id，字段id） 
    GroupId, 组ID（包括信息组ID，表格组ID）

CreateDate, 创建日期
Creater,创建者
LastmodifyDate, 最后修改日期
LastmodifyUser, 最后修改者

)
